About dotstail

The Problem

As we - Front-end Devs all know that Css text-overflow: ellipsis gives result as a single line only, which is annoying sometimes when the outcome wanted is a mutiple line ellipsis.

What is dotstail?

Dotstail is a repo that help you to custom a cross browser text-overflow: ellipsis for mutiple lines by using JavaScript and a little CSS line.

How to use

Set Up

Call the action:
dotstail.setDot({ Options });

Options:

Call the element:
'element': 'class/id'

Numbers of letter wanted to be cut off (optional):
'letter': number (default: 0)

Numbers of line wanted to be ellipsis (optional):
'lines': number (default: 1)

Important CSS Required

Element's line-height needed to work (must be a em unit):
line-height: ...em;

Examples:

Full Option:

dotstail.setDot({

'element': '.dotstail-1',

'letter': 2,

'lines': 2

});

Remove 'letter' Option:

dotstail.setDot({

'element': '.dotstail-2',

'lines': 3

});
